全栈工程师（2021年3月 ~ 至今）

  1. 员工自助平台

    • 背景： 公司内部核心业务，这是一个基于微信公众号开发的 H5 移动端项目，旨在整合公司内部的常用业务，推动业务流程线上化、标准化，降低运营成本。
    • 任务：
      • 完成前端架构设计与核心功能开发（技术栈：MUI框架+原生JS+jQuery），主导后端 C#+Oracle 数据库搭建，实现前后端一体化开发，确保数据交互高效性与扩展性。
      • 集成微信公众号 JS-SDK 能力，实现员工身份鉴权、手机拍照验证、扫码识别等移动端原生交互，攻克多终端适配与性能瓶颈。
    • 行动：
      • 基于 MUI 组件库开发标准化页面，通过动态 DOM 渲染与异步加载技术，实现H5页面在iOS/Android/PC三端界面与交互一致性。
      • 利用 JS-SDK 调用微信原生能力：通过 wx.getLocation 实现外勤打卡地理围栏校验；结合 wx.chooseImage 与图像压缩算法，将报销凭证上传体积减少60%；集成 wx.scanQRCode 自动解析工单编号，减少人工输入错误。
      • 使用 C# 和 Oracle 数据库开发后端，确保数据结构的合理性。
    • 结果： 自员工自助平台上线覆盖80%+员工，日均活跃用户超2000人，单业务模块平均处理时长从15分钟缩短至3分钟，年节约打印、人力成本约35万元。关键事务（如跨部门审批）流转效率提升70%，用户满意度达94%。

  2. IOP 后台管理系统

    • 背景： 企业原有业务系统分散于人资、行政、环安等8个独立模块，存在数据孤岛、跨系统操作复杂、响应式支持不足。
    • 任务：
      • 基于 H+ 框架和 Bootstrap 构建 PC 端一体化运营中台，集成20+核心业务流程，打通数据壁垒，实现跨部门协作标准化与决策数据可视化。
      • 设计响应式布局方案，实现不同电脑的全分辨率适配，解决IE11/Chrome/Firefox多浏览器兼容性问题。
      • 重构部分交互逻辑，通过组件化开发（如可视化图表引擎）提升功能复用率，降低二次开发成本。
    • 行动：
      • 基于 Bootstrap 栅格系统与 H+ 主题模板构建响应式布局。
      • 使用 C# 开发.net 后台，与 Oracle 数据库进行数据的读写。
      • 采用 LocalStorage+IndexedDB 双缓存策略，使低频访问数据加载速度提升40%。
    • 结果： 系统上线后支撑公司90%+日常运营事务，业务单据处理效率提升65%，跨部门协作耗时减少50%。年节约跨系统对接成本百万元以上。

  3. 轨迹追踪系统

    • 背景： 随着公司园区规模扩大，员工安全管理与行为分析需求日益迫切。传统人工巡检模式效率低、实时性差，需通过技术手段实现员工从入厂到离厂的全程轨迹追踪。
    • 任务：
      • 前端框架的搭建与选型，实现轨迹动态渲染、实时监控看板、数据分析模块。
      • 后端接口编写，设计适合员工-摄像头-点位相关联的表结构。
    • 行动：
      • 基于 SoybeanAdmin 中后台模板二次开发，使用 Vue3 + Vite5 + TypeScript + UnoCSS 等最新技术。
      • 调用海康接口获取员工点位，基于 Leaflet 绘制员工轨迹，实现轨迹的播放与呈现。通过 L.geoJSON 划定禁区，触发越界告警并记录事件日志。
      • 通过 ECharts 生成员工热力图、高频路径分析报表，辅助优化园区动线设计。
    • 结果： 实现员工轨迹全流程数字化监控，告警响应时间缩短至3秒内。管理效率提升60%，减少安保人力成本。

  4. 劳务返费小程序

    • 背景： 公司劳务返费流程依赖纸质表单与人工审核，存在信息错漏率高、进度不透明、发放周期长等问题。为提升劳务工操作便捷性与公司结算效率，急需实现全流程线上化闭环管理。
    • 任务：
      • 主导小程序前端架构设计与开发，基于uni-app框架与uView组件库搭建多端兼容界面，支持H5、Android、iOS三端一致性交互体验。
      • 实现劳务工身份信息动态表单填写、手写签名采集（Canvas+第三方插件）、返费协议PDF生成与加密下载、历史记录多维度查询（按时间/状态筛选）等核心功能。
    • 行动：
      • 采用 uView 动态表单组件构建可配置化信息填写页面，支持身份证OCR识别（集成百度云AI能力）与数据实时校验，降低人工输入错误率。
      • 开发电子签名模块：基于 Canvas 实现手写签名轨迹捕获，结合 signature_pad 插件优化笔触流畅度，生成 Base64 图像并嵌入 PDF 模板。
      • 通过 uni-file-system 与 jsPDF 库动态生成标准化返费协议 PDF，支持本地缓存及一键分享至微信/邮箱，提升数据可追溯性。
    • 结果： 小程序上线后覆盖7家劳务公司超5000名劳务工，返费申请处理时效从3天缩短至2小时内，信息准确率提升至99.8%，每年减少纸质成本15万元。支持7×24小时自助查询，人工咨询量下降80%。

  5. 立铠之窗

    • 背景： 公司存在40+业务系统分散、入口不统一问题，员工需频繁记录系统地址，跨平台操作效率低下。为提升办公协同效率，需构建统一门户整合高频系统、自定义工具及常用办公服务，实现一站式访问与管理。
    • 任务：
      • 前端实现可拖拽排序的自定义导航页，集成多维度办公工具（PDF处理、考勤查询等），设计可扩展的快捷菜单架构。
      • 后端设计支持用户个性化配置的数据表结构，开发工具类功能接口，实现权限隔离与数据安全。
    • 行动：
      • 基于 Vue3 + Vite5 + TypeScript 构建页面，搭配 Pinia 状态管理，路由控制使用 Vue Router
      • 采用 VueDraggablePlus 实现系统网址自定义排序，数据持久化至后端
      • 基于 pdf-lib 开发PDF合并/拆分功能小工具，采用 browser-image-compression 实现纯前端图片压缩小工具。
    • 结果： 上线后日均访问量2000+人次，系统寻址时间降低85%，工具模块节省日均1.5小时/人事务性工作时间。

  6. 招募辅助插件

    • 背景： 公司招募员工后需在政府指定网站手动填报大量登记信息，传统人工输入效率低、易出错，且重复操作耗时耗力。需通过技术手段实现数据自动填充，提升登记效率与准确性。
    • 任务：
      • 开发 Chrome 浏览器插件，实现身份证触发自动填充、跨域数据请求、表单动态绑定。
      • 设计插件与公司数据库的交互逻辑，确保数据安全传输与隐私保护。
    • 行动：
      • 基于 Chrome Extension Manifest V3 规范开发，使用 TypeScript + Vite 构建插件核心逻辑，通过 Content Scripts 注入目标页面并操作 DOM 实现表单自动填充。
      • 利用 Service Worker 处理插件后台逻辑，调用公司数据库接口，基于身份证号查询员工信息，使用 DES 加密 保障数据传输安全。
      • 通过 MutationObserver 监听页面动态加载的表单元素，结合 CSS 选择器 精准匹配栏位，实现复杂结构表单的智能填充。
    • 结果： 单条信息填报时间从 15 分钟缩短至 10 秒内，错误率降低 95%，人力成本减少 50%。

    • 任务：
      • 负责项目的前端页面开发，使用 Vue 框架实现用户界面的构建与交互逻辑。使用 swiper 等插件美化页面。
      • 使用 axios 配合后端进行数据交互，实现数据的获取、提交和更新。
      • 通过数据可视化（ECharts）分析用户行为，驱动优惠券精准投放策略。
    • 行动：
      • 使用Vue.js框架构建前端页面，实现用户界面和交互逻辑。利用vue-router进行页面导航和跳转，优化路由映射和组件懒加载。骨架屏技术减少白屏时长，提升用户体验感知。集成swiper插件，实现首页商品Banner的动态轮播效果。

  2. 工资单小程序

    • 背景： 公司原有工资单通过邮件或纸质方式发放，存在查询不便、信息滞后、易泄露等问题。为提升员工体验、实现薪酬数据移动化管理，决定开发微信小程序，集成工资查询、历史记录、明细下载等功能，降低沟通成本并提高数据安全性。
    • 任务：
      • 负责小程序前端开发，基于微信原生框架和Vant组件库搭建界面，确保界面美观与操作一致性。
      • 实现微信登录鉴权、工资数据动态渲染、PDF生成与下载等核心功能。
      • 与后端协作设计高效接口，优化数据交互性能与安全性。解决多端适配问题，确保安卓/iOS系统兼容性。
    • 行动：
      • 使用Vant组件库搭建标准化页面（如表格、弹窗、下拉刷新），结合WXSS实现响应式布局。
      • 开发动态渲染模块，将后端返回的工资数据分类展示（基本薪资、津贴、扣款项等），支持按月份筛选。
      • 与后端定义高效接口规范，采用分页加载与懒加载优化大数据量场景性能。集成微信文件系统API，实现工资单PDF生成、本地缓存及一键下载功能。
