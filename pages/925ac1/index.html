mui滚动条与textarea自身滚动冲突

使用 mui 的区域滚动后，在区域滚动内有 textarea 多行内容会无法滚动，原因可能是被父级 div 的滚动效果覆盖了

解决方法：

window.addEventListener('touchmove', function(e) {  
    var target = e.target;  
    if (target && target.tagName === 'TEXTAREA') {//textarea阻止冒泡  
        e.stopPropagation();  
    }
}, true)
